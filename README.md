# Salah Junior Dev — Portfolio

Personal portfolio website built to showcase my work as a full-stack web developer, UI/UX designer, and graphic designer based in Yaoundé, Cameroon.

Live site → [salahjuniordev.vercel.app](https://salahjuniordev.vercel.app)

---

## About

I started building websites at 17 out of pure curiosity. Three years later it became my craft. This portfolio is where I bring everything together — the projects I've shipped, the services I offer, and a bit of who I am behind the screen.

I work with NGOs, logistics companies, and brands across Central Africa, turning ideas into digital products that actually work.

---

## What's Inside

- **Hero section** with animated headline cycling through my roles
- **About** — background, personal info, and interests
- **Services** — what I offer and how I work
- **Portfolio** — selected projects with tags and live links
- **Pricing** — three plans from starter to full-scale
- **Testimonials** — feedback from clients
- **Blog** — thoughts and articles
- **Contact** — direct form to reach me

---

## Tech Used

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Rubik)
- Cloudinary for image hosting
- Deployed on Vercel

---

## Features

- Fully responsive — mobile, tablet, and desktop
- Bilingual (English / French) with a one-click language toggle
- Custom animated cursor
- Smooth scroll navigation
- Animated counters, flip cards, and scroll reveal effects
- Custom page loader
- SEO optimized with Open Graph and Twitter Card meta tags
- Schema markup for Google structured data

---

## Getting Started

No build tools or dependencies needed. Just clone and open.

```bash
git clone https://github.com/salahjuniordev/portfolio.git
cd portfolio
```

Then open `index.html` in your browser — that's it.
